# Archives

This folder contains older, reference-only files that are not part of the active runtime path.

Its purpose is to preserve experiments, previous setup assets, and older helper scripts that may still be useful when revisiting earlier design decisions.

## Files and Subfolders

|------|---------|

| `list_repos.py` | Older helper for listing configured repositories. |

| `settings/` | Archived settings-related notes. |
