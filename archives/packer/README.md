# Archived Packer Files

This folder contains archived Packer and autoinstall files used for earlier VM image creation work.

These files are kept for reference and are not part of the current default install flow.

## Files

| File | Purpose |
|------|---------|
| `ubuntu-server.pkr.hcl` | Packer template for building an Ubuntu Server image. |
| `autoinstall.yaml` | Ubuntu autoinstall configuration used by the archived Packer workflow. |
