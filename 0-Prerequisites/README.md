# Prerequisites

This folder contains assets used before the main installer runs.

Its purpose is to prepare the base VM or host image so the main `setup/` scripts can assume a cleaner starting point.

## Files and Subfolders

| Item | Purpose |
|------|---------|
| `linux-golden-image/` | Scripts used while preparing the reusable Ubuntu "golden image" before Docker and the agent stack are installed. |

Read more in folder `0-Prerequisites/linux-golden-image`.
